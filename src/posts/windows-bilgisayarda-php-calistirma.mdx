#### Merhabalar, bugün sizlere evinizdeki veya ofisinizdeki bilgisayarda PHP'yi nasıl kurup kullanıcağınızı anlatıcam. Keyifli okumalar :)

İlk olarak [Buradan](https://www.apachefriends.org/tr/index.html "Tıklayınız") xampp isimli programı bilgisayarımıza kuruyoruz.

Ardından kurulu olduğu klasöre girip aşağıda size vermiş olduğum görseldeki dosyayı yani **xampp-control.exe**'yi bulup açıyoruz. ![xampp ile php çalıştırma](/images/xampp.png)

Ve ekranda aşağıdaki gibi bir pencere açılacak görselde de işaretlediğim gibi **Apache** mödülünün karşısındaki Start butonuna basıp sunucumuzu başlatmamız gerekmektedir.

![xampp server nasıl başlatılır](/images/xampp-control.png)

**Apache**'mizi başlattıktan sonra ise tekrardan **xampp**'ın kurulu olduğu klasöre giriyoruz görselde gördüğünüz **htdocs** adlı klasöre girip PHP dosyalarımızı bu klasörün içinde oluşturuyoruz.

![](/images/htdocs.png)

Size bahsettiğim adımları yaptıktan sonra **[http://localhost](http://localhost)** adresinden hertürlü php uygulamalarınızı çalıştırabilirsiniz.
